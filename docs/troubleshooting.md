# Troubleshooting

Apologies in advance if you have any trouble using Corridor Key Toolbox.

Please read this page first, and if the solutions here don't solve your problem, please [post an issue](https://github.com/latenitefilms/CorridorKeyToolbox/issues) on GitHub and we'll do our best to solve it.

---

## It doesn't work?

Make sure you're running macOS Tahoe 26 on a modern Apple Silicon Mac with as much RAM as you can.
